# Overview

Healthcare HMS Platform is a **scalable, distributed healthcare management system** designed to digitalize hospital operations.

It enables:
- Patient lifecycle management
- Doctor scheduling & availability tracking
- Appointment orchestration
- Medical record storage
- Lab report management
- Billing & insurance handling
- Real-time healthcare coordination

Built with a **microservices architecture**, it ensures high scalability, fault isolation, and production-grade reliability.
